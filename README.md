# Full-Stack Developer Portfolio - MERN Stack

A modern, dynamic portfolio application featuring a robust backend architecture, an interactive AI assistant, and a full-featured admin dashboard for real-time content management.

## 🚀 Overview

This project is a high-performance web application built with the MERN stack (MongoDB, Express, React, Node.js). It showcases advanced backend integration, secure authentication, and a dynamic frontend that synchronizes site-wide updates instantly.

## 🧠 Backend Excellence (Core Focus)

Having 2.9 years of professional experience primarily focused on backend engineering, I've implemented a scalable and secure server-side architecture for this portfolio:

- **State-of-the-Art REST APIs**: Custom-built Node.js/Express server handling endpoints for seamless data orchestration.
- **Secure Authentication**: JWT-based authentication system for administrative access.
- **Dynamic Data Modeling**: MongoDB with Mongoose schemas for complex entities (Profile, Projects, Skills, Experience, Education).
- **Automation & Integration**: 
  - Automated PDF generation via backend triggers.
  - Third-party API integrations (Stripe, Directus).
- **Performance Optimization**: Efficient database query indexing and optimized response middleware.

## 💻 Technical Stack

### Backend
- **Node.js & Express**: Core server framework.
- **MongoDB & Mongoose**: NoSQL database for flexible content management.
- **JWT**: Secure authentication and session management.
- **CORS & Helm**: Enhanced security and cross-origin resource sharing.

### Frontend
- **React.js**: Modular component-based UI.
- **React Router**: Seamless client-side navigation.
- **Lucide React**: Modern, scalable iconography.
- **React Type Animation**: Dynamic interactive elements.

## ✨ Key Features

- **Admin Dashboard**: Full CRUD management for projects, skills, and professional experience.
- **Dynamic Home Page**: "Code Editor" and "Terminal" views driven by real-time backend data.
- **AI-Powered Contact Assistant**: A bot knowledge-base synchronized with the database for instant, accurate information.
- **Responsive Resume**: Live-updating skill proficiency bars and interactive experience timeline.

## 🛠️ Installation & Setup

### Prerequisites
- Node.js (v16+)
- MongoDB (Local or Atlas)

### Server Setup
```bash
cd server
npm install
# Create .env file with MONGODB_URI and JWT_SECRET
npm start
```

### Client Setup
```bash
cd client
npm install
npm start
```
